# HisouSangokushiZero2_1
## 概要
悲愴三国志Zero2 バージョン1の開発プロジェクト  
開発技術：C#,Xaml,Uno Platform(webassembly)
## ゲーム内容
悲愴感をコンセプトにしている制作中ゲームの悲愴三国志が完成するか不安なので作った簡易版、悲愴三国志Zeroの４つ目のバージョン  
「反董連合」と「蜀之南征」の２つのシナリオ  
シンプルな内政パラメータと人物パラメータ、多めの勢力数と都市数  
登場する勢力は全てプレイ可能、勝利条件は全土統一ではなく、勢力ごとに設定されている

## 攻略テクニック
前線管理と人物の維持、内政値と首都の防衛が重要  
首都を落とされると人物が大量に戦死してしまう  
内政値を維持しないとどんどん行動できなくなっていってしまう  
しかし行動できる手数とリソースには限りがある、勢力と状況ごとに優先すべきことを考えよう  
防衛専念を上手く生かそう、クリック連打が最適手じゃない  
防衛専念では攻めてきた敵の人物が戦死しやすい、人物の層が戦況に直結するのでこれは重要  
逆に、敵の防衛専念に突っ込まないように、敵の休み状態を戦略に生かそう  
  
CPUチートなし、プレイヤーと同条件で、プレイヤーの入力の前に行動を確定させている  
CPUも前線の軍圧を感じ取り人物を配置する、出兵先の選定も前線管理を考えている、ただし精度低め
